## Описание
Это приложение позволяет отслеживать личные расходы с возможностью добавления записей о расходах, фильтрации по категории и подсчета суммы расходов за выбранный период. Данные сохраняются в формате JSON.

## Как использовать
1. Убедитесь, что у вас установлен Python.
2. Запустите приложение с помощью команды python rash.py.
3. Введите сумму расхода.
4. Выберите категорию.
5. Введите дату в формате YYYY-MM-DD.
6. Нажмите кнопку «Добавить расход» для сохранения записи.
7. Используйте фильтр для отображения записей по категории.
8. Введите даты начала и окончания периода для подсчета общей суммы расходов.
